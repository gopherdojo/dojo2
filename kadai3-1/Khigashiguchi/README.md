# kadai3-1
## 課題内容
- タイピングゲームを作ろう
    - 標準出力に英単語を出す（出すものは自由）
    - 標準入力から1行受け取る
    - 制限時間内に何問解けたか表示する

## 実装フロー
- [ ] 予め出す「英単語」を定義しておく
- [ ] 標準出力に「英単語」を出す
- [ ] 標準入力を待つ
    - [ ] 標準入力があっている場合は次の問題に行く
    - [ ] 標準入力が間違っていた場合には間違っている旨を伝えた上再度標準入力を待つ
    - [ ] 制限時間内に問題が解けなかった場合は次に問題に行く
- [ ] 最終問題が終わったら何問正解したかを表示する
